# Voice Assistant Project

A simple voice assistant built using Python that can respond to voice commands and fetch weather information using an API.

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Requests
- dotenv

## Features
- Voice input
- Text-to-speech output
- Weather information
- Basic commands

## Setup Instructions

1. Install dependencies:
   pip install -r requirements.txt

2. Create a `.env` file:
   WEATHER_API_KEY=your_api_key_here

3. Run the program:
   python main.py

## Important
Do NOT upload your `.env` file to GitHub.

Make sure `.env` is added to `.gitignore`.
