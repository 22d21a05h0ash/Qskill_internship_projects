# NumPy Practice Project

This project contains practice programs and exercises using NumPy for numerical computing.

## Technologies Used
- Python
- NumPy
- Jupyter Notebook

## Project Description
Includes examples of array creation, manipulation, indexing, and mathematical operations.

## Files
- numpy.ipynb
- numpy_project.py

## How to Run

### For Python File
python numpy_project.py

### For Notebook
jupyter notebook numpy.ipynb

- Arrays
- Reshaping
- Broadcasting
- Aggregations
- Axis operations

## Learning Outcome
