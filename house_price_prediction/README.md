# House Price Prediction

This project predicts house prices based on features like square feet, bedrooms, bathrooms, and location using Machine Learning.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Project Description
A regression model is trained on housing data to estimate prices for new houses.

## Files
- houseprice_prediction.py
- house_data.csv

## How to Run

1. Install required libraries:
   pip install numpy pandas scikit-learn matplotlib

2. Run the program:
   python houseprice_prediction.py

## Output
The model predicts house prices and displays accuracy.

## Learning Outcome
- Data preprocessing
- Linear regression
- Model training & testing
- Data visualization
